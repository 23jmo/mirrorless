# Orb Avatar: Replace Memoji with ElevenLabs Orb + Streaming TTS

## Context

The current avatar system uses pre-recorded Memoji video loops (6 states) + 13 scripted response videos with per-sentence TTS fetching. It's complex, glitchy (blob URL crashes), and requires ~5MB of video assets. We're replacing the entire visual + audio pipeline with:

1. **ElevenLabs UI Orb** ŌĆö 3D WebGL sphere (`@elevenlabs/cli` component) that reacts to audio volume
2. **Streaming TTS** ŌĆö ElevenLabs REST streaming endpoint instead of per-sentence fetch
3. **Emotion-tagged responses** ŌĆö Claude prefixes each response with `[emotion:X]`, parsed on frontend
4. **Context-aware floating** ŌĆö Orb drifts to different screen positions based on state

## Decisions from Interview

| Decision | Choice |
|---|---|
| Orb component | ElevenLabs UI Orb (`@elevenlabs/cli`, Three.js WebGL) |
| Orb placement | Context-aware: corner when idle, center when greeting, near products when showing |
| Scripted videos | Kill entirely ŌĆö no video files |
| TTS approach | ElevenLabs REST streaming (`/stream` endpoint) via backend proxy |
| STT | Keep Deepgram (unchanged) |
| Emotion source | Claude streams `[emotion:X]` prefix tag, parsed on frontend |
| Emotion palette | 4 states: idle (#F0F0F5), neutral (#F5E6A0), proud (#4A6FA5), teasing (#FFF3B0) |
| Orb states | Full state machine: idle ŌåÆ listening ŌåÆ thinking ŌåÆ speaking ŌåÆ idle |
| Visual style | Glass sphere with glow (ElevenLabs Orb is 3D WebGL with inner glow) |

**New file: `frontend/src/lib/streaming-tts.ts`**

Replace the per-sentence `ElevenLabsTTS` class with a streaming approach using `fetch` + `ReadableStream` + `MediaSource` API.

```typescript
// Key pattern from ElevenLabs docs:
const response = await fetch(`${apiUrl}/api/tts/stream`, {
  method: "POST",
  headers: { "Content-Type": "application/json" },
  body: JSON.stringify({ text }),
});

const mediaSource = new MediaSource();
const audio = new Audio();
audio.src = URL.createObjectURL(mediaSource);

mediaSource.addEventListener("sourceopen", () => {
  const sourceBuffer = mediaSource.addSourceBuffer("audio/mpeg");
  const reader = response.body!.getReader();

  // Read chunks and append to source buffer
  async function pump() {
    const { done, value } = await reader.read();
    if (done) {
      mediaSource.endOfStream();
      return;
    }
    sourceBuffer.appendBuffer(value);
    sourceBuffer.addEventListener("updateend", pump, { once: true });
  }
  pump();
});

audio.play();
```

**Volume extraction for Orb (from ElevenLabs SDK source):**
```typescript
// Route audio through AudioContext + AnalyserNode
const audioContext = new AudioContext();
const source = audioContext.createMediaElementSource(audio);
const analyser = audioContext.createAnalyser();
source.connect(analyser);
analyser.connect(audioContext.destination);

// Calculate volume (ElevenLabs SDK pattern from VoiceConversation.ts):
function getOutputVolume(): number {
  const frequencyData = new Uint8Array(analyser.frequencyBinCount);
  analyser.getByteFrequencyData(frequencyData);
  let volume = 0;
  for (let i = 0; i < frequencyData.length; i++) {
    volume += frequencyData[i] / 255;
  }
  volume /= frequencyData.length;
  return Math.max(0, Math.min(1, volume));
}
```

```typescript
function parseEmotionTag(text: string): { emotion: MiraEmotion; cleanText: string } {
  const match = text.match(/^\[emotion:(\w+)\]\s*/);
  if (match) {
    const emotion = match[1] as MiraEmotion;
    const validEmotions: MiraEmotion[] = ["idle", "neutral", "proud", "teasing"];
    return {
      emotion: validEmotions.includes(emotion) ? emotion : "neutral",
      cleanText: text.slice(match[0].length),
    };
  }
  return { emotion: "neutral", cleanText: text };
}
```

## Two Approaches

### Approach 1: REST Streaming (Simpler -- Recommended for Mirrorless)

**Endpoint:** `POST https://api.elevenlabs.io/v1/text-to-speech/{voice_id}/stream`

Uses standard `fetch` with `response.body.getReader()` to read audio chunks as they arrive, then feeds them into a `MediaSource` + `SourceBuffer` for browser playback. The audio format is MP3 (browser-native, no manual decoding needed).

### Approach 2: WebSocket Streaming (Lower Latency, More Complex)

**Endpoint:** `wss://api.elevenlabs.io/v1/text-to-speech/{voice_id}/stream-input?model_id=eleven_multilingual_v2&output_format=pcm_44100`

Uses a WebSocket with a specific message protocol (BOS/text/flush/EOS messages). Audio arrives as base64-encoded PCM16 chunks in JSON, which you decode manually and schedule for gapless playback via `AudioContext.createBufferSource()`.

---

## 1. WebSocket Streaming -- Full Details

### URL Format
```
wss://api.elevenlabs.io/v1/text-to-speech/{voice_id}/stream-input?model_id={model}&output_format=pcm_44100
```

Regional servers: `wss://api.us.elevenlabs.io/...`, `wss://api.eu.residency.elevenlabs.io/...`

### Authentication (3 options)
1. `xi_api_key` field in the first (BOS) JSON message
2. `authorization` field in the first message (`"Bearer <token>"`)
3. `xi-api-key` as a query parameter in the URL

### Message Protocol

**Step 1 -- BOS (Beginning-of-Stream):**
```json
{
  "text": " ",
  "voice_settings": {
    "stability": 0.5,
    "similarity_boost": 0.8,
    "style": 0,
    "use_speaker_boost": true
  },
  "generation_config": {
    "chunk_length_schedule": [120, 160, 250, 290]
  },
  "xi_api_key": "YOUR_API_KEY"
}
```

**Step 2 -- Send text (repeat as needed, text MUST end with a space):**
```json
{
  "text": "Hello, how are you today? ",
  "try_trigger_generation": true
}
```

**Step 3 -- Flush (force-generate remaining buffered text):**
```json
{
  "text": " ",
  "flush": true
}
```

**Step 4 -- EOS (End-of-Stream, text is empty string):**
```json
{
  "text": ""
}
```

### Server Responses

**Audio chunk:**
```json
{
  "audio": "base64_encoded_pcm16_data",
  "alignment": {
    "chars": ["H", "e", "l", "l", "o"],
    "charStartTimesMs": [0, 50, 100, 150, 200],
    "charDurationsMs": [50, 50, 50, 50, 100]
  }
}
```

**Final message (generation complete):**
```json
{
  "isFinal": true
}
```

### Complete Working Code -- WebSocket + AudioContext (PCM playback)

```typescript
export class ElevenLabsWebSocketTTS {
  private ws: WebSocket | null = null;
  private audioContext: AudioContext | null = null;
  private nextPlayTime = 0;
  private speaking = false;
  private readonly sampleRate = 44100;

  private voiceId: string;
  private apiKey: string;
  private modelId: string;

  constructor(voiceId: string, apiKey: string, modelId = "eleven_multilingual_v2") {
    this.voiceId = voiceId;
    this.apiKey = apiKey;
    this.modelId = modelId;
  }

  async speak(
    text: string,
    onStart?: () => void,
    onEnd?: () => void,
  ): Promise<void> {
    if (!this.audioContext || this.audioContext.state === "closed") {
      this.audioContext = new AudioContext({ sampleRate: this.sampleRate });
    }
    if (this.audioContext.state === "suspended") {
      await this.audioContext.resume();
    }

    this.nextPlayTime = this.audioContext.currentTime;
    this.speaking = true;

    return new Promise<void>((resolve) => {
      const wsUrl =
        `wss://api.elevenlabs.io/v1/text-to-speech/${this.voiceId}/stream-input` +
        `?model_id=${this.modelId}` +
        `&output_format=pcm_${this.sampleRate}`;

      const ws = new WebSocket(wsUrl);
      this.ws = ws;

      ws.onopen = () => {
        // 1. BOS message with auth + voice settings
        ws.send(JSON.stringify({
          text: " ",
          voice_settings: {
            stability: 0.5,
            similarity_boost: 0.8,
            style: 0,
            use_speaker_boost: true,
          },
          generation_config: {
            chunk_length_schedule: [120, 160, 250, 290],
          },
          xi_api_key: this.apiKey,
        }));

        onStart?.();

        // 2. Send the actual text (must end with space)
        ws.send(JSON.stringify({
          text: text + " ",
          try_trigger_generation: true,
        }));

        // 3. Flush remaining
        ws.send(JSON.stringify({ text: " ", flush: true }));

        // 4. EOS
        ws.send(JSON.stringify({ text: "" }));
      };

      ws.onmessage = (event) => {
        const data = JSON.parse(event.data);

        if (data.audio) {
          this.playPCMChunk(data.audio);
        }

        if (data.isFinal) {
          const remainingTime = Math.max(0, this.nextPlayTime - this.audioContext!.currentTime);
          setTimeout(() => {
            this.speaking = false;
            onEnd?.();
            resolve();
          }, remainingTime * 1000);
        }
      };

      ws.onerror = (err) => {
        console.error("[WS-TTS] WebSocket error:", err);
        this.speaking = false;
        onEnd?.();
        resolve();
      };

      ws.onclose = () => { this.ws = null; };
    });
  }

  /** Decode base64 PCM16 and schedule for gapless playback */
  private playPCMChunk(base64Audio: string): void {
    if (!this.audioContext) return;

    // Base64 -> binary
    const binaryString = atob(base64Audio);
    const bytes = new Uint8Array(binaryString.length);
    for (let i = 0; i < binaryString.length; i++) {
      bytes[i] = binaryString.charCodeAt(i);
    }

    // PCM16 signed 16-bit little-endian -> Float32 [-1.0, 1.0]
    const pcm16 = new Int16Array(bytes.buffer);
    const float32 = new Float32Array(pcm16.length);
    for (let i = 0; i < pcm16.length; i++) {
      float32[i] = pcm16[i] / 32768;
    }

    // Create AudioBuffer
    const audioBuffer = this.audioContext.createBuffer(1, float32.length, this.sampleRate);
    audioBuffer.copyToChannel(float32, 0);

    // Schedule playback
    const source = this.audioContext.createBufferSource();
    source.buffer = audioBuffer;
    source.connect(this.audioContext.destination);

    const startTime = Math.max(this.nextPlayTime, this.audioContext.currentTime);
    source.start(startTime);
    this.nextPlayTime = startTime + audioBuffer.duration;
  }

  stop(): void {
    if (this.ws) { this.ws.close(); this.ws = null; }
    this.speaking = false;
    if (this.audioContext) { this.audioContext.close(); this.audioContext = null; }
  }

  isSpeaking(): boolean { return this.speaking; }
}
```

### Complete Working Code -- REST + MediaSource (MP3 playback)

```typescript
export class ElevenLabsStreamingTTS {
  private apiUrl: string;
  private audio: HTMLAudioElement | null = null;
  private mediaSource: MediaSource | null = null;
  private speaking = false;
  private generation = 0;

  constructor(apiUrl: string) {
    this.apiUrl = apiUrl;
  }

  async speak(
    text: string,
    onStart?: () => void,
    onEnd?: () => void,
  ): Promise<void> {
    const gen = ++this.generation;

    try {
      this.speaking = true;
      onStart?.();

      const resp = await fetch(`${this.apiUrl}/api/tts/speak`, {
        method: "POST",
        headers: { "Content-Type": "application/json" },
        body: JSON.stringify({ text }),
      });

      if (this.generation !== gen) { this.speaking = false; return; }
      if (!resp.ok) throw new Error(`TTS proxy error: ${resp.status}`);
      if (!resp.body) throw new Error("No response body for streaming");

      if (typeof MediaSource !== "undefined" && MediaSource.isTypeSupported("audio/mpeg")) {
        await this.playWithMediaSource(resp.body, gen, onEnd);
      } else {
        // Fallback: buffer entire response
        const blob = await resp.blob();
        if (this.generation !== gen) { this.speaking = false; return; }
        await this.playBlob(blob, onEnd);
      }
    } catch (err) {
      console.warn("[TTS] Streaming failed:", err);
      this.speaking = false;
      onEnd?.();
    }
  }

  private playWithMediaSource(
    body: ReadableStream<Uint8Array>,
    gen: number,
    onEnd?: () => void,
  ): Promise<void> {
    return new Promise<void>((resolve) => {
      const mediaSource = new MediaSource();
      this.mediaSource = mediaSource;

      const audio = new Audio();
      audio.src = URL.createObjectURL(mediaSource);
      this.audio = audio;

      mediaSource.addEventListener("sourceopen", async () => {
        const sourceBuffer = mediaSource.addSourceBuffer("audio/mpeg");
        const reader = body.getReader();

        const appendQueue: Uint8Array[] = [];
        let streamDone = false;
        let isAppending = false;

        const processQueue = () => {
          if (isAppending || appendQueue.length === 0) return;
          if (sourceBuffer.updating) return;
          isAppending = true;
          const chunk = appendQueue.shift()!;
          sourceBuffer.appendBuffer(chunk);
        };

        sourceBuffer.addEventListener("updateend", () => {
          isAppending = false;
          if (streamDone && appendQueue.length === 0) {
            if (mediaSource.readyState === "open") {
              mediaSource.endOfStream();
            }
          } else {
            processQueue();
          }
        });

        try {
          while (true) {
            const { done, value } = await reader.read();
            if (done || this.generation !== gen) break;
            appendQueue.push(value);
            processQueue();
          }
        } catch (err) {
          console.error("[TTS] Stream read error:", err);
        }
        streamDone = true;
        processQueue();
      });

      audio.onended = () => {
        this.cleanup();
        this.speaking = false;
        onEnd?.();
        resolve();
      };

      audio.onerror = () => {
        this.cleanup();
        this.speaking = false;
        onEnd?.();
        resolve();
      };

      audio.play().catch(() => {
        this.cleanup();
        this.speaking = false;
        onEnd?.();
        resolve();
      });
    });
  }

  private playBlob(blob: Blob, onEnd?: () => void): Promise<void> {
    return new Promise<void>((resolve) => {
      const url = URL.createObjectURL(blob);
      const audio = new Audio(url);
      this.audio = audio;
      const done = () => {
        URL.revokeObjectURL(url);
        this.speaking = false;
        onEnd?.();
        resolve();
      };
      audio.onended = done;
      audio.onerror = done;
      audio.play().catch(done);
    });
  }

  stop(): void {
    this.generation++;
    if (this.audio) {
      this.audio.onended = null;
      this.audio.onerror = null;
      this.audio.pause();
      this.audio = null;
    }
    this.cleanup();
    this.speaking = false;
  }

  private cleanup(): void {
    if (this.audio?.src) URL.revokeObjectURL(this.audio.src);
    this.mediaSource = null;
  }

  isSpeaking(): boolean { return this.speaking; }
}
```

```python
            error_body = await resp.aread()
            await client.aclose()

                detail=f"ElevenLabs error ({resp.status_code}): {error_body[:200]}")

        await client.aclose()
        raise HTTPException(status_code=502, detail=f"Request failed: {exc}")

        try:
            async for chunk in resp.aiter_bytes(chunk_size=4096):
                yield chunk
        finally:
            await resp.aclose()
            await client.aclose()
```

## Comparison Table

| Factor | REST Streaming | WebSocket Streaming |
|--------|---------------|---------------------|
| **Complexity** | Lower -- fetch + MediaSource | Higher -- WS lifecycle + PCM decode |
| **Time to first audio** | ~500ms-1s | ~200-500ms |
| **Best for** | Full text available upfront | Token-by-token from LLM |
| **Audio format** | MP3 (browser-native) | PCM (manual decode needed) |
| **API key** | Stays on backend (proxy) | Exposed in browser (or needs proxy) |
| **Cancellation** | AbortController on fetch | ws.close() |

## Key Gotchas

1. **API key security**: WebSocket from browser exposes your key. Use the REST proxy approach, or the single-use token endpoint.
2. **MediaSource + Safari**: `MediaSource.isTypeSupported("audio/mpeg")` can be false on older Safari. Always include a blob fallback.
3. **PCM sample rate must match**: `output_format=pcm_44100` in the URL must match `new AudioContext({ sampleRate: 44100 })`. Mismatch causes pitch distortion.
4. **Text must end with space**: In WebSocket mode, every `text` field (except EOS) must end with a trailing space.
5. **chunk_length_schedule**: Default `[120, 160, 250, 290]` means: 1st chunk generates after 120 chars, 2nd after 160, etc. Use `flush: true` for short text.
6. **AudioContext requires user gesture**: Browsers block autoplay. Call `audioContext.resume()` inside a click handler.
7. **Your current backend is "fake streaming"**: `StreamingResponse(iter([resp.content]))` buffers the entire response in memory first. Change to `httpx` streaming with `stream=True` for real streaming.
