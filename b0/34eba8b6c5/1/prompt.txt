# Plan: Frontend Chat Demo for Mira

## Context

We need a quick frontend page to test Mira interactively via the browser. The backend Socket.io events are wired up and working (verified via CLI tests). This page replaces voice input with a text chat box for now (Deepgram STT comes later). We'll work in the main worktree, not the agent-work worktree.

## Approach

Add a single new page at `/chat` with a chat interface that connects to the backend via the existing Socket.io client.

### New file: `frontend/src/app/chat/page.tsx`

A single-page chat UI with:

**Layout (top to bottom):**
1. **Header** — "Chat with Mira" + session controls (Start / End)
2. **Chat messages** — scrolling message area showing:
   - Mira's streamed speech (assembled from `mira_speech` chunks into complete messages)
   - User's sent messages
   - Product cards when `tool_result` arrives (title, price, source, image, rating)
   - Session events (started, ended, recap)
3. **Gesture buttons** — row of buttons: 👍 👎 ⬅️ ➡️ (thumbs up/down, swipe left/right)
4. **Text input** — input field + send button (sends as `voice` transcript via `mirror_event`)

**Socket.io flow:**
- On mount: `socket.connect()`, emit `join_room` with user ID
- Start button: emit `start_session`
- Send message: emit `mirror_event` with `{type: "voice", transcript: "..."}`
- Gesture buttons: emit `mirror_event` with `{type: "gesture", gesture: "thumbs_up"}` etc.
- End button: emit `end_session`
- Listen: `mira_speech` (stream chunks into current message), `tool_result` (render product cards), `session_ended` / `session_recap`

**Reuses existing:**
- `@/lib/socket` — Socket.io client (`frontend/src/lib/socket.ts`)
- Inline styles matching existing project convention (dark theme, purple accents)

**User ID:** Text input at top to paste a real UUID from DB, or defaults to `"chat-demo-user"` for testing without DB.

### Files to modify

| File | Action |
|------|--------|
| `frontend/src/app/chat/page.tsx` | **Create** — full chat page |

That's it. One file. No other changes needed.
