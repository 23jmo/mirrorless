# Add "Start Session" Button to Mirror UI

## Context

Currently, starting a Mira session requires navigating to the `/chat` page, pasting a user_id, and clicking "Start Session" there — while the mirror page passively waits. This is a clunky workaround. The user wants the mirror page to be self-contained: one button to start Mira directly from the mirror UI.

The backend already handles everything via a `start_session` socket event (in `backend/main.py:93-111`). The mirror page just needs to emit it.

## File to Modify

`frontend/src/app/mirror/page.tsx`

## Changes

### 1. Add `isStarting` loading state

```typescript
const [isStarting, setIsStarting] = useState(false);
```

### 2. Add `startSession` callback

Emits the existing `start_session` socket event. Sets loading state so the button shows feedback. Resets on error.

```typescript
const startSession = useCallback(() => {
  if (!userId || isStarting || sessionActive) return;
  setIsStarting(true);
  socket.emit("start_session", { user_id: userId });
}, [userId, isStarting, sessionActive]);
```

### 3. Clear `isStarting` when session activates

In the existing `handleSessionStarted` callback (and the fallback mira_speech detector), add `setIsStarting(false)` so the loading state clears once the session is actually running.

Also add a `session_error` listener to reset `isStarting` on failure.

### 4. Render the button (pre-session only)

Show a centered "Start Session" button when:
- `userId` is present (from URL query param)
- `sessionActive` is false
- Button shows "Starting..." with disabled state while `isStarting` is true

If no `userId` in URL, show a message like "No user — scan QR or add ?user_id= to URL"

Inline styles (consistent with the project's existing pattern — no CSS framework).

```
┌─────────────────────────────┐
│                             │
│     [camera feed behind]    │
│                             │
│      ┌─────────────────┐    │
│      │  Start Session   │   │  ← centered, semi-transparent bg
│      └─────────────────┘    │
│                             │
│  MIRROR-A1          (badge) │
└─────────────────────────────┘
```
